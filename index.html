Portfolio

Hover through some of the projects I have worked on.

01

Heart Disease Classification

A combination of Jupyter Notebook, Scikit-Learn, MatplotLib, Pandas, NumPy, Seaborn and a Kaggle provided dataset allowed me to create a classification model that helps predict whether a patient has heart disease given certain clinical parameters.

02

California Housing Regression

A combination of Jupyter Notebook, Scikit-Learn, MatplotLib, Pandas, NumPy, Seaborn and a Kaggle provided dataset allowed me to create a regression model that helps predict the price of a home in California based on it's location and some other factors.

03

Dog Breed Image Convolution

A combination of Google Colab, Tensorflow, MatplotLib, Pandas, NumPy and a Kaggle provided dataset allowed me to create a predictive model that helps predict whether the breed of a dog from just the image.

04

Autonomous RC Car

Used an Arduino Uno with a L293D Motor Drive Shield Expansion Board to control SG90 9G Servo Motors, alongside a ultrasonic distance sensor (HC-SR04) to guide a RC car automatically stopping at obstacles and turning.

05

Basketball Mini Hoop Shot Tracker

Used an Arduino Uno with a HC-SR04 (Ultrasonic distance sensor) and the MPU-6050 (accelerometer and gyro) to track a basketball shot, and whether it's a make or miss which would then be displayed on a LCD screen.

06

Hand Gesture Controlled RC Car

Used an Arduino Nano's with a L298n (Dual Motor Driver) and the NRF24L01 (radio Communicator) to have a link between the hand control and RC car.

07

PC LED Lights

Used an Arduino Uno with potentiometer's, switch's, and transistors to control LED light strips within the PC, displaying both static and dynamic color ranges.

08

Inventory Management System

Used Java with the Netbeans IDE to create a terminal based inventory management system allowing small businesses to add, remove, and track inventory along with creating invoices.
